CartMate JavaScript Project

  • Category: Mobile App design
  • Project date: April 2023
  • Live Site URL: CartMate Netlify
  • Project Solution URL: CartMate Github

This project involved creating a mobile app using JavaScript that allows users to add grocery items to a list, remove items by tapping on them, and toggle between two modes, peaches&cream and kiwi, by using a toggle button. The app enables users to easily manage their grocery list on their mobile device.
