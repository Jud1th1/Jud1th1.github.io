The task was to create a homepage/landing page for Naples, Florida that would be easy to navigate for both desktop and mobile users. The page includes a dropdown navigation bar, links to pages from the current Naples webpage, hover-over menu images, social media icons, and a Google Translate feature in the footer. The page was also designed with accessibility in mind to enable screen readers and keyboard users to navigate the sections.
